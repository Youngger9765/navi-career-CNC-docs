## M1：報名金流

### 完整流程

```
學員                                系統                              行政
 │                                   │                                 │
 ├── 選擇班別（A / A+B / A+B+C）──▶ 顯示價格 + 名額                  │
 │   選擇銷售期間（早鳥/原價）       即時計算名額池                    │
 │                                   │                                 │
 ├── 登入/註冊 ──────────────────▶ 建立帳號                          │
 │   （信箱 + 密碼 + 驗證信）        此帳密 = 課程系統帳密             │
 │                                   │                                 │
 ├── 填寫個人資料 ──────────────▶ 驗證必填欄位                       │
 │   姓名、信箱、手機                │                                 │
 │   地址（台灣/海外）               │                                 │
 │   已有牌卡類型                    │                                 │
 │                                   │                                 │
 ├── 輸入折扣碼 ────────────────▶ 驗證折扣碼有效性                   │
 │   推薦人姓名                      計算折扣金額                      │
 │                                   │                                 │
 ├── 選擇付款方式 ──────────────▶ 串接綠界 ECPay                     │
 │   信用卡付清/分期/ATM/超商        │                                 │
 │                                   │                                 │
 ├── 選擇發票類型 ──────────────▶ 電子發票/捐贈/統編                 │
 │                                   │                                 │
 ├── 勾選確認規則 + 送出 ──────▶ 導向綠界付款頁                     │
 │                                   │                                 │
 │   ┌─ 付款成功 ──────────────▶ 報名成功頁                         │
 │   │                               寄出成功信（課前調查、時程、LINE）│
 │   │                               自動開立發票 ────────────────▶ 確認金流
 │   │                               │                                上標籤
 │   │                               │                                 │

 │                                   寄出失敗通知信                    │
 └                                   └                                 └
```

## M2：行銷頁面 & 開課設定

### 完整流程

```
營運人員                                     系統
 │                                            │
 ├── 設定年度開課日期 ──────────────────▶ 建立梯次
 ├── 確認講師時間 ─────────────────────▶ 設定行事曆（自動寄發給老師）
 │                                            │
 ├── 一鍵開設當梯次 ───────────────────▶ 建立：
 │                                            ├ 會議連結
 │                                            ├ LINE 群組加入連結
 │                                            ├ 報名商品（→ M1）
 │                                            └ 報名成功信模板
 │                                            │
 ├── 編輯行銷頁面（CMS）──────────────▶ 標準化模板：
 │   ├ 課程名稱 / 開課單位                    ├ 文字、影片、圖片、GIF
 │   ├ 課程介紹區                              ├ 圖片跑馬燈
 │   ├ 常見問題（下拉選單）                    ├ 里程碑數字動畫
 │   └ 商品選單                                └ 搜集信箱 CTA
 │                                            │
 └── 上線 ────────────────────────────▶ 行銷頁面公開
                                               商品選單連動 M1 報名
```

### 三大行銷入口

| 入口 | 流程 | 與報名的連動 |
|------|------|-------------|
| **一日班** | 報名 → 寄出牌卡 → 上課 → 課後調查 | 發放折扣碼 → M1 |
| **說明會** | Accupass 報名 → 行前提醒 → 上線 → 會後信（回放+折扣碼）→ 優惠截止信 | 折扣碼 → M1 |
| **學長姐聊聊** | 確認學長姐 → 上架（自介+預約連結）→ 學員預約 → 聊聊 | 優惠碼 → M1 |

### 商品類別

| 類型 | 購買方式 | 品項 |
|------|---------|------|
| 課程類 | **需註冊帳號** | 一日班、大培訓班、CDA、進修課程、線上課程 |
| 電商類 | 非會員可購買 | 周邊產品（另外做，不在本模組） |

## M3：運課管理

### 完整流程

```
                    ┌──────────────────────────────────────────────────┐
                    │                    運課管理                        │
                    │                                                    │
  M1 報名成功 ────▶│  確認入班 ──▶ A班正課(6天) ──▶ B班活動(14次)     │──▶ M4 認證
                    │      │                                             │
                    │      ├── 寄送包裹（講義、牌卡）                    │
                    │      └── 課前通知（Email + LINE）                  │
                    │                                                    │
                    └──────────────────────────────────────────────────┘
```

### A 班正課（6 天循環）

```
每天：
  課前 ──▶ 學員完成課前作業（Figma 連結）
  上課 ──▶ Google Meet 線上課程
  課後 ──▶ 行政上架補課影片（YouTube 嵌入）
         ──▶ 行政公告下次作業
         ──▶ 點名紀錄
```

### B 班活動時間線

```
  ┌─ A班實體見面會（訂小樹屋、調查出席、訂餐點）
  │
  ├─ 學長姐諮詢體驗（平分名單 → 確認 → 公告給雙方）
  │
  ├─ 團督 ×7（個案演練 24hr、公益個案登記、上架補課影片）
  │
  ├─ 公益諮詢（需求調查 → 名單媒合 → 公告給雙方）
  │
  └─ 實體結業式（訂小樹屋、訂餐點、調查出席）
```

### 學員繳交項目（B 班）

- 附件 A：總表
- 附件 B：同意書
- 附件 C：個案報告
- 附件 D：個案諮詢回饋
- 附件 E：團督/個督心得

| 文件繳交 | Google 文件/表單 | **取代** | 自建上傳區 |

## M5：畢業後 CRM

### 完整流程

```
M4 認證通過 ──▶ 會員 CRM ──▶ 續證（2.5年後自動提醒）
                   │
                   ├──▶ 特約諮詢師申請 ──▶ 資深特約晉升
                   │
                   └──▶ 推薦返現計畫 ──▶ M1 報名折扣
```

### 會員 CRM

```
會員資料：
├── 基本資料
├── CNC 證書狀態：有效 / 過期 / 期限
└── 特殊身份標籤（含期限）：
    特約、資深特約、特約講師、
    學長姐聊聊、學長姐諮詢體驗、講師、督導

可見性層級：
├── 公開：全世界（官網展示）
├── 學員：全體學員
└── 特殊：僅特殊身份
```

### 續證流程

```
系統                              學員                        負責人
 │                                 │                           │
 ├── 自動提醒（2.5年期限）──▶ 看到「我要續證」              │
 │                                 │                           │
 │                    上傳續證檢核資料 ──▶                     │
 │                                 │                           │
 │   IF 時數不符 ──▶ 無法送出      │                           │
 │   IF 時數符合 ──▶ 送出申請      │                           │
 │                                 │                           │
 │                    繳費（串 M1 金流）──▶                    │
 │                                 │                           │
 │                                 │              確認檢核項目  │
 │                                 │              點選「續證完畢」
 │                                 │                           │
 ├── 證書狀態 → 有效 ─────────────────────────────────┘
 └── 更新證書
```

### 特約諮詢師申請

```
學員                              系統                        行政
 │                                 │                           │
 ├── 提出申請特約考核 ─────▶ 辨識證書有效性                  │
 │                                 IF 無效 → 導去續證          │
 │                                 │                           │
 ├── 上傳特約檢核資料 ─────▶ │                              │
 │                                 │                           │
 │   ◀── 考核通過 + 回饋 ────┤                    助教確認資訊：
 │                                 │                    姓名、信箱、手機、
 │                                 │                    合約期間、特約等級、
 │                                 │                    官網資訊（文字+照片+LINE）
 │                                 │                           │
 │                                 │              負責人 → 提供名單給營運、行銷
 │                                 │                           │
 │                                 │              營運：準備合約（點點簽）
 │   ◀── 簽署合約 ────────┤              學員簽回
 │                                 │              製作月結表、上架預約表單、SUPER8
 │                                 │              負責人勾選合約簽回
 │                                 │              官網上架 → 追蹤 → 勾選完畢
 │                                 │              點選「晉升作業完畢」→ 上標籤
 └                                 └                           └
```

### 暫不納入系統（導流外部工具）

> USB 插槽原則：斷掉的地方先導到外部工具，後續迭代再收回。

- **SUPER8** → 導流
- **點點簽** → 導流
- **月結表** → 營運先用目前方式
