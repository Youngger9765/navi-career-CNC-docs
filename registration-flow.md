# CNC 報名系統 — 流程與 QA 測試表

## 系統概述

CNC 職涯導航師培訓的**線上報名與付款系統**，取代原本 Google Forms + 人工對帳流程。

```
課程介紹 → 選擇梯次 → 填寫個資 → 折扣碼 → 確認金額 → 付款 → 完成報名
```

## 系統特色

- **即時名額管理**：資料庫鎖定防超賣
- **彈性定價**：早鳥折扣 + 折扣碼（自動取最優）
- **綠界金流**：信用卡（含分期）、ATM、銀行轉帳
- **自動狀態更新**：付款完成自動更新報名狀態
- **Email 通知**：報名確認 + 付款成功自動寄信
